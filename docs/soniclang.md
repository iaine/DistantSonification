## Sonic Languages

I have been exploring machine learning with sonic languages.

### Manifold

This is a language and implementation to listen to a concept of the [manifold](https://github.com/iaine/manifold) within machine learning. 

This is a Python desktop application that will need building. 

### MagL

This is a language that describes the loss function within the machine learning training process. It has two forms: one that represents loss as magnitude and the second is the cross entropy. 

It runs from a notebook for exploration. 

[MagL](https://github.com/iaine/magL)
