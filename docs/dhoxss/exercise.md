## Exercise

In this exercise, you can either create a design for a sonification or use tools to sonify your own data. 

You have options to think about. 

1. Design a Sonification


2. Create an Open Sonification

In this option, you can create a sonification using your own data (or you can use ours). 

You could also use tools such as TwoTone to sonify the data.

If you are interested in the audio mapping, then you could try the [audio mapping](sound-map.html) tool.
